Fundamentals

Variables

Containers for storing data values. let name = "Neil"; // String let age = 20; // Number let isStudent = true; // Boolean console.log(name, age, isStudent);

Control Structures

Direct the flow of the program based on conditions or loops. let grade = 85; if (grade >= 50) { console.log("Pass"); } else { console.log("Fail"); }

Functions

Reusable blocks of code that perform a task. function greet(name) { console.log("Hello, " + name); } greet("Neil"); // Hello, Neil greet("Kalvin"); // Hello, Kalvin greet("Sumalinog"); // Hello, Sumalinog

Data Structures

Ways to organize and store data efficiently. let fruits = ["Apple", "Banana", "Cherry"]; fruits.push("Date"); console.log(fruits);
